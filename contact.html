To ask for help...

If you have questions about the GSPBox, then please send an email to the mailing list : gspbox 'dash' support 'at' groupes 'dot' epfl 'dot' ch.

To report a bug...

If you think you have found a bug in the gspbox, then please check out the issues page and add it there. The bug tracker contains a list of all current issues with the GSPBox and all feature requests for the future.
