Usage

[gd,filtertype] = gsp_jtv_design_can_dual( g,filtertype );

Description

Inputs parameters:
g : cell array of time-vertex filters filtertype : Filter domain (ts,js,ts-array,js-array)
Ouputs parameters:
g : cell array of canonical dual time-vertex filters filtertype : Filter domain (ts,js)

This function returns the canonical dual of the time-vertex filterbank g
