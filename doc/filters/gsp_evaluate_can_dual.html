GSP_EVALUATE_CAN_DUAL - Evaluate the canonical dual filterbank

Usage

hcoeff = gsp_evaluate_can_dual( g,val )

Description

Inputs parameters:
g : cell array of filters val : column vectors of values tol : tolerance
Ouputs parameters:
s : Matrix of value

This function computes the value of the canonical dual of a filterbank g at the point specified in val. The function returns a matrix. Each column is the output of one dual filter.
