GSP_STATIONARITY_COV - Covariance matrix from graph stationary data

Usage

C = gsp_stationarity_cov(X)

Input parameters

X Data (M x N matrix)

Output parameters

C Covariance matrix (M x M)

Description

This function estimates the covariance from the data. Every sample has the same expected average.

References:

N. Perraudin and P. Vandergheynst. Stationary signal processing on graphs. arXiv preprint arXiv:1601.02522, 2016.
